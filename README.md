# Nook

Nook Lab 的公共代码集合。

## Structure

| Directory | Description |
|-----------|-------------|
| `skills/` | SKILL.md files — AI agent instructions for various tasks |
| `mcp/` | MCP servers — standalone tools callable by any MCP-capable agent |
| `web/` | Web projects and frontend applications |
| `other/` | Miscellaneous tools and experiments |

## Repos

This repo consolidates the following archived repositories:
- nook-skills
- nook-mcp
- nook-card
- nook-tv-cover
- yunduan_web1.0

See each subdirectory's README or SKILL.md for details.
