## 2. 当前页面结构

- `/`：首页封面、近期视频、公众号文章、内容形态、课程入口、联系入口
- `/about`：关于船长的角落与云端心流
- `/contact`：联系合作
- `/wechat/[YYMMDD]`：本地公众号文章详情页
- `/admin`：后台管理
- `/editor`：文章编辑器

## 3. 公众号文章如何同步

当前首页会自动读取本地公众号文章库：

- `D:/MMDBB_vault/20 Areas/AI_Tech/公众号文章库`

维护方式：

1. 用 WeChat Converter 插件在 vault 中生成公众号 Markdown。
2. 文件放在上述目录。
3. 文件名保持当前规则，例如：`260503_AI越强越需要边界_公众号.md`。
4. 确保 frontmatter 中有 `title` / `publish_title` 和 `summary` / `publish_summary`。
5. 正文中的本地图片放在 `assets/...`。
6. 启动网站后首页会读取最近文章，并生成 `/wechat/260503` 这类站内页面。

实现文件：

- `lib/local-wechat-posts.ts`
- `app/wechat/[slug]/page.tsx`

图片处理：

- 跳过通用公众号头图 `20260502142235164.jpg`
- 优先取正文真实配图
- 本地图片会复制到 `public/wechat-assets/`

## 4. B 站视频如何同步

当前近期视频在：

- `components/themes/HomeDefault.tsx`

封面本地目录：

- `public/bilibili-covers/`

同步流程：

1. 提供 B 站视频链接或 BV 号。
2. 通过 B 站接口读取标题和封面。
3. 下载封面到 `public/bilibili-covers/`。
4. 更新首页视频配置。
5. 运行 `npm run build` 检查。

当前为了避免 B 站封面外链防盗链，首页只引用本地封面。

## 5. 后台文章发布

后台发布仍然保留：

1. 打开 `/admin`
2. 输入管理密码
3. 点击“写文章”
4. 在编辑器中填写标题、正文、摘要、分类、封面等
5. 保存并发布

后台文章主要走数据库；公众号文章目前走本地 Markdown 自动读取。
