# 云端心流 / 船长的角落 网站需求简报 v2

## 1. 项目定位

本项目是 **船长的角落** 官网，背后公司主体为 **云端心流**。

网站不是传统企业官网，也不是单纯博客，而是一个以 AI 内容为核心的品牌阵地：

- 展示频道与公司品牌
- 呈现 B 站视频
- 沉淀公众号文章
- 为未来图文、播客、课程、工作坊保留空间
- 提供商务合作、课程合作与咨询入口

## 2. 当前品牌关系

- 前台主品牌：船长的角落
- 公司 / 后台主体：云端心流
- 频道方向：AI 工具、趋势观察、创作实践、效率方法、知识管理

当前首页主文案：

- 主标题：船长的角落
- Slogan：在内容的海面上，找到 AI 的航向

## 3. 核心用户

网站主要面向：

1. 对 AI 工具、AI 趋势、AI 应用感兴趣的个人用户。
2. 关注 AI 提效、内容创作、知识管理的职场人士。
3. 从 B 站、公众号等平台认识频道的用户。
4. 潜在课程学员。
5. 潜在企业客户或合作伙伴。

## 6. 跨设备继续工作需求

为了保证在公司、家里或其他场景都能继续工作，需要：

1. 项目代码同步到私有 GitHub 仓库。
2. README 说明清楚本地开发、内容来源和常用命令。
3. 内容维护文档说明清楚 B 站视频和公众号文章如何同步。
4. 新电脑需要能访问公众号文章库目录，或修改 `lib/local-wechat-posts.ts` 中的路径。

私有仓库：

- `https://github.com/captain-nook/yunduan_web1.0`
