# 船长的角落 网站框架 v2

## 1. 项目定位

**船长的角落** 是云端心流旗下的 AI 内容频道官网。

网站目标不是单纯展示公司信息，而是建立一个长期可运营的内容阵地，用于：

- 品牌展示
- B 站视频呈现
- 公众号文章沉淀
- 未来图文 / 播客扩展
- 课程入口与后续转化
- 商务合作与联系入口

## 2. 当前核心原则

1. 首页首先像一个“频道封面”，不是传统信息堆叠首页。
2. “船长的角落”是前台主品牌，“云端心流”是公司与后台主体。
3. 视觉风格保持浅色、明亮、清爽、蓝青主调。
4. 标题参考 Logo 取色：深蓝主标题、黄色点缀、青色副标题。
5. 内容优先于营销，品牌感优先于销售感。
6. V1 做内容展示和基础转化，不做复杂交易系统。

## 6. 内容维护策略

### 6.1 B 站视频

短期维护方式：代码配置 + 本地封面。

流程：

1. 提供 B 站链接或 BV 号。
2. 读取标题和封面。
3. 下载封面到 `public/bilibili-covers/`。
4. 更新首页视频配置。
5. 运行构建检查。

### 6.2 公众号文章

短期维护方式：本地 Markdown 自动读取。

流程：

1. 用 WeChat Converter 在 vault 中生成 Markdown。
2. 文件放入公众号文章库。
3. 确保 frontmatter 有标题和摘要。
4. 启动网站后首页自动显示。

### 6.3 后台文章

后台文章仍通过 `/admin` 和 `/editor` 发布，主要走数据库。
