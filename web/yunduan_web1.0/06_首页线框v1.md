# 船长的角落 首页线框 v2

## 1. 页面目标

首页要在第一屏完成三件事：

1. 让用户记住“船长的角落”这个频道。
2. 让用户感受到清爽、明亮、有设计感的品牌气质。
3. 让用户能继续进入视频、文章、关于我们或联系合作。

首页当前不是传统企业官网，也不是纯博客列表页，而是一个**内容频道封面页**。

---

## 3. 视觉组件规则

### 按钮

- 主按钮：深色实心
- 次按钮：白底浅描边
- 圆角胶囊

### 卡片

- 白底或半透明白底
- 轻边框
- 轻阴影
- 大圆角
- Hover 时轻微上浮

### 标签

- 圆角胶囊
- 使用蓝青色系

### 图片

- 头像使用 `public/captain-avatar.png`
- Logo 使用 `public/captain-logo.png`
- B 站封面使用 `public/bilibili-covers/`
- 公众号图片使用 `public/wechat-assets/`

## 4. 响应式规则

### 桌面端

- Hero 左右布局
