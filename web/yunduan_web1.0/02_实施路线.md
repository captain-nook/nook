## 跨设备继续工作流程

在任意新电脑：

```bash
git clone https://github.com/captain-nook/yunduan_web1.0.git
cd yunduan_web1.0
npm install
npm run dev
```

如果需要公众号文章同步，还需要让新电脑能访问：

```text
D:/MMDBB_vault/20 Areas/AI_Tech/公众号文章库
```

如果路径不同，修改：

```text
lib/local-wechat-posts.ts
```
